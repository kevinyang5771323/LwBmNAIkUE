# 社区养老管理系统简介 java790

## 项目概述

社区养老管理系统是一款基于前后端分离架构的信息管理系统，旨在为社区养老提供便捷的服务管理、老人信息登记、服务发布、志愿者招募等功能。

## 技术栈

- 后端开发语言：Java
- 主要框架：SpringBoot（后端）、Vue（前端）
- 数据库：MySQL
- 开发工具：IDEA、VSCode
- 开发环境：JDK 8、Node.js 12

## 功能模块

- 老人信息管理：实现对社区老人基本信息和健康状况的登记、查询、更新和删除。
- 服务记录：发布服务信息，志愿者可对服务进行接收和完成状态处理。
- 服务配置：对服务项目进行增加、删除、修改和查询。
- 活动管理：制定、查询、更新和删除活动计划。
- 志愿者管理：发布招募信息，并进行志愿者的增删改查操作。
- 志愿者请假管理：记录和查询志愿者的请假信息。
- 系统用户管理：对系统用户进行增删改查，以及密码重置等操作。
- 角色管理：配置角色信息，并对角色权限进行管理。

## 系统角色

系统包含以下角色：
- 老人亲属
- 志愿者
- 管理员

## 运行环境

- 服务器：推荐使用Linux操作系统
- 数据库：MySQL 5.7 或更高版本
- 服务器硬件：至少 2GB 内存，2 核 CPU

## 部署步骤

1. 准备所需服务器和网络环境。
2. 部署 MySQL 数据库，并进行配置。
3. 部署后端服务，确保 JDK 8 和 Node.js 12 已安装。
4. 部署前端服务，构建 Vue 项目并部署至 Web 服务器。
5. 进行系统测试和配置。
